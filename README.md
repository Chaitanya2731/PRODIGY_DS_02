# 📊 Task 02 - Exploratory Data Analysis

## 🎯 Objective
Analyze the dataset and visualize passenger information.

## 🛠 Tools Used
Python, Pandas, Matplotlib, Seaborn, Google Colab

## 📌 What I Did
- Loaded the dataset  
- Checked columns and missing values  
- Filled missing values in Age and Fare  
- Removed Cabin column  
- Created charts for Age, Gender and Class  
- Compared Age and Fare using scatter plot  

## 📊 Results
- Most passengers are between 20–40 years old  
- Majority of passengers are in 3rd class  
- Male passengers are more than female passengers  
- Fare distribution is uneven  
- Age and Fare do not show strong relationship
